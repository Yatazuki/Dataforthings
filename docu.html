Game Functions

Snake Game
A classic snake game implementation with modern graphics and controls.
Key functions: - initGame(): Initializes the game board and snake - moveSnake(): Handles snake movement and collision detection - generateFood(): Creates new food at random positions - checkCollision(): Detects collisions with walls or self
Tic-tac-toe
A two-player implementation of the classic Tic-tac-toe game.
Key functions: - checkWinner(): Checks for winning combinations - makeMove(): Handles player moves - resetGame(): Resets the game board - updateBoard(): Updates the visual representation
Memory Game
A card matching memory game with scoring system.
Key functions: - shuffleCards(): Randomizes card positions - flipCard(): Handles card flipping - checkMatch(): Verifies matching pairs - updateScore(): Updates player score
Click Speed Test
A test to measure and improve clicking speed.
Key functions: - startTest(): Initiates the speed test - countClicks(): Tracks click count - calculateScore(): Computes final score - saveHighScore(): Stores best results

Authentication Functions

User Authentication
Handles user registration, login, and session management.
Key functions: - registerUser(): Creates new user accounts - loginUser(): Authenticates users - logoutUser(): Handles user logout - checkSession(): Verifies active sessions

Data Management
Handles all database operations and data persistence.
